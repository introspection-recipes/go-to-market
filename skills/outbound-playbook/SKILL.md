# Outbound Playbook

Use this skill when drafting or reviewing outbound sales messages.

## Rules

- Run do-not-send checks before drafting.
- Personalize from verified account, role, relationship, or product usage signals.
- Keep outreach concise and specific.
- Include rationale for the rep, not in the customer-facing message.
- Never send directly; route to Slack approval.

## Output

Return the draft, evidence used, personalization rationale, risks, and the approval message.
